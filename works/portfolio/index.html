ポートフォリオサイトです。
職業訓練での卒業制作として作成しました。訓練にて学んだデザイン作成、コーディング技術に加えてGSAPによるアニメーションを取り入れたサイトとなっております。

WEB SITE
POINT
初めてGSAPを使用したスクロール連動のアニメーション作成を行いました。レスポンシブ対応させるのがなかなか大変でしたが、とても勉強になりました。作品紹介ページの画像スクロール部分（スクロールバー）も現状出来る限りのカスタマイズを行いました。
FVの背景と次のセクションへの背景アニメーションをthree.jsを使用して描画処理を作成しました。前へ進む、移動している感を表現したかったのですが、正直なところ満足のいく出来ではなく、独学での技術の使用をチャレンジしてみたという結果となりました。
