天気予報アプリ

HTML JavaScript Responsive アプリ

天気予報APIを使ったアプリ。
JavaScriptのfetch機能を使ってAPI連携を実装しました。

WEB SITE
POINT
Javascriptのfetch機能によるapi連携を実装しました。気象庁の天気予報apiを使用しています。
localStorage機能を使用したユーザーの選択状態の保存、ダークモード切り替えの実装も行いました。
ScrollHintを使用してSP表示にて一部スクロールヒントを実装しました。
