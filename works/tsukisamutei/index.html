源流の宿 月寒亭

HTML JavaScript

非日常を楽しむ山あいの宿、ひとときを贅沢に過ごす架空の温泉旅館サイトです。
JavaScriptを学習後、訓練校での課題として作成しました。PCでの閲覧のみを想定としています。

WEB SITE
POINT
ターゲットは、30代以上の男女、少し贅沢に休日を過ごしたい方としています。
日本語フォントはFOT-筑紫Aオールド明朝 Pr6N、欧文フォントはAdobe Garamond Proを使用し、サイトに高級感を持たせています。
JavaScriptを使用し、ナビゲーションリンクのスムーススクロール、スクロール位置依存のヘッダー切り替え、クリックによる画像の切り替え等を実装しています。
