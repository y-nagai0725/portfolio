cafe azito

HTML JavaScript

非日常を味わう。海の景色、匂いを間近に感じられるカフェサイト。
訓練校でのWEBデザイン学習の課題として作成しました。PCでの閲覧のみを想定しています。

WEB SITE
POINT
ターゲットは、女性、20～30代、おしゃれな空間で過ごしたい方をしています。
日本語フォントは游明朝、欧文フォントはSabonを使用し、女性向け、おしゃれさを表現しています。
Swiper.jsを使用してスライダーを実装し、お店の内装や外観、お店から見える風景等の画像を見られるようにしています。
